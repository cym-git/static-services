## 静态文件服务器

该服务是一个静态文件服务器，类似于 `http-server` 工具，可以在任意目录下启动该静态文件服务器，并且把当前目录作为文件服务器

该服务器支持资源缓存，资源压缩

## Version

1.1.2

## Example

可以直接在需要作为静态服务的目录打开您的 `Terminal`，输入 `static-services` 自动启动静态资源服务器，默认端口是 8080

```sh
static-services -d 指定静态文件根目录 -p 指定端口号  -o 指定监听的主机
```
